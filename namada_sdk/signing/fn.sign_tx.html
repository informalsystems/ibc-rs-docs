Expand description

Sign a transaction with a given signing key or public key of a given signer. If no explicit signer given, use the default. If no default is given, Error.

It also takes a second, optional keypair to sign the wrapper header separately.

If this is not a dry run, the tx is put in a wrapper and returned along with hashes needed for monitoring the tx on chain.

If it is a dry run, it is not put in a wrapper, but returned as is.
