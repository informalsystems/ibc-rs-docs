Expand description

A serialization method to provide to namada_tx::Signed, such that we may sign serialized data.

This is a higher level version of SignableBytes.

Required Associated Types§

source

type Output: SignableBytes

A byte vector containing the serialized data.

source

type Hasher: 'static + StorageHasher

The hashing algorithm to use to sign serialized data with.

Required Methods§

source

fn as_signable(data: &T) -> Self::Output

Encodes data as a byte vector, with some arbitrary serialization method.

The returned output must be deterministic based on data, so that two callers signing the same data will be signing the same Self::Output.
