Expand description

Abstract of consensus state information used by the validity predicate to verify new commits & state roots.

Users are not expected to implement sealed::ErasedPartialEqConsensusState. Effectively, that trait bound mandates implementers to derive PartialEq, after which our blanket implementation will implement ErasedPartialEqConsensusState for their type.
