Expand description

Trait that provides access to the components of an encrypted transaction output.

Implementations of this trait are required to define the length of their ciphertext field. In order to use the trial decryption APIs in this crate, the length must be either ENC_CIPHERTEXT_SIZE or COMPACT_NOTE_SIZE.
