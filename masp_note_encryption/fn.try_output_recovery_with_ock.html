Recovery of the full note plaintext by the sender.

Attempts to decrypt and validate the given shielded output using the given ock. If successful, the corresponding note and memo are returned, along with the address to which the note was sent.

Implements part of section 4.19.3 of the Zcash Protocol Specification. For decryption using a Full Viewing Key see try_output_recovery_with_ovk.
