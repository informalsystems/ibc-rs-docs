Module string_encoding

source
Expand description

Namada’s standard string encoding for public types.

We’re using bech32m, a format with a human-readable, followed by base32 encoding with a limited character set with checksum check.

To use this encoding for a new type, add a HRP (human-readable part) const below and use it to impl string_encoding::Format for YourType.
