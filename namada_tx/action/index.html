namada_tx

Module action

source
Expand description

Actions applied in a transaction.

Tx actions are used to indicate from tx to VPs the type of actions that have been applied by the tx to simplify validation (We can check that the storage changes are valid based on the action, rather than trying to derive the action from storage changes). When used, the kind is expected to written to under temporary storage (discarded after tx execution and validation).
