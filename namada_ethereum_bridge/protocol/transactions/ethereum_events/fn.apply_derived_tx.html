Expand description

Applies derived state changes to storage, based on Ethereum events which were newly seen by some consensus validator(s).

For events which have been seen by enough voting power (>= 2/3), extra state changes may take place, such as minting of wrapped ERC20s.

This function is deterministic based on some existing blockchain state and the passed events.
