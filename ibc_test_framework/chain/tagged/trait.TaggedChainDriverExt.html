Expand description

A ChainDriver may be tagged with a Chain tag in the form MonoTagged<Chain, ChainDriver>.

It would implement the TaggedChainDriverExt trait to provide tagged version of the chain methods.

The tagged chain driver methods help ensure that the ChainDriver methods are used with the values associated to the correct chain.

Tagged version of ChainDriver::assert_eventual_escrowed_amount_ics29.

Assert that a wallet should eventually have escrowed the amount for ICS29 fees of a given denomination. Legacy ICS29 will escrow recv_fee + ack_fee + timeout_fee while more recent versions will escrow max(recv_fee + ack_fee, timeout_fee).
