Types for information about a chain wallet.

Structs§

  • TestWallets
    A collection of wallets used for testing. We use an explicit struct instead of a generic HashMap so that the retrieval of a specific wallet can always succeed. We shouldn’t need more than the wallets listed here for testing purposes.
  • Wallet
    A wallet containing the information about the ID, address, and also the public/private key information in the form of SigningKeyPair.
  • WalletAddress
    Newtype wrapper for the address a wallet corresponds to.
  • WalletId
    Newtype wrapper for a wallet ID as identified by the chain and relayer.
