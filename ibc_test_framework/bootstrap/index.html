Module ibc_test_framework::bootstrap

source ·
Expand description

Helper functions for setting up test cases in an imperative way.

Normal test authors should have no need to call functions provided by the bootstrap module, as they are implicitly called by the framework constructs.

Advanced test authors with needs for more flexibility can call functions in the bootstrap module directly, so that they have more control of when exactly new chains and relayers should be spawned.
