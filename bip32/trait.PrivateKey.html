Provided Methods§

source

fn derive_tweak( &self, chain_code: &ChainCode, child_number: ChildNumber, ) -> Result<(PrivateKeyBytes, ChainCode)>

Derive a tweak value that can be used to generate the child key (see [derive_child]).

The chain_code is either a newly initialized one, or one obtained from the previous invocation of derive_tweak() (for a multi-level derivation).

Warning: make sure that if you are creating a new chain_code, you are doing so in a cryptographically safe way. Normally this would be done according to BIP-39 (within [ExtendedPrivateKey::new]).
